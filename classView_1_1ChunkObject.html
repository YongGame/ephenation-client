Detailed Description

Manage visible attributes of a chunk. This is part of the View. A chunk is 32x32x32 blocks, which can't be used for drawing as it is. It is converted into two types: visible blocks and special objects. It is the responsibility of this class to take the chunk data and compute what can be drawn from the chunk.
