Detailed Description

A common base class for libRocket dialogs.

The actual dialog content is defined in external files in dialogs/ *.rml (Rocket Markup Language), in a format similar to html. Inherit from Rocket::Core::EventListener to make it possible to register class for Rocket event callbacks. There is a stack of documents, to able to have more than one active document at the same time. Though only the document at the top of the stack will be shown. The stack has to be common between all managers inheriting from this class, which is why it is declared as static.
