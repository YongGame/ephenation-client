Detailed Description

Manage buffers and shaders for the various render passes.

All drawing stages are controlled from this class. The main activities are: 1. A Frame Buffer Object to use as a target for the first rendering. 2. Call the functions that update the images in the FBO 3. Project the final result to the screen.
